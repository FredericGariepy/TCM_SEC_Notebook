## `if elif else`
```
if not True:
  print('I will never be executed')
elif 0b10 == 2:
  if not False:
    print('I will be executed')
    print('Me too!')
  else:
    print('I will never print..')
else:
  print('Me too, I will never print..')
```
## ternary
`print('I won\'t print') if 1 > 0b10 else print ('let the world see me shine.')`
